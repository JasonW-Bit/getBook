# 报告文档目录

本目录存放项目的各种报告、统计和分析文档。

## 目录结构

```
docs/reports/
├── optimization/     # 优化相关报告
├── project/         # 项目相关报告
├── code/            # 代码相关报告
└── statistics/      # 统计数据（未来使用）
```

## 报告分类

### 优化报告 (`optimization/`)

- **OPTIMIZATION_REPORT.md** - 最新优化报告（2025-12-14）
  - 爬虫功能优化
  - 数据处理优化
  - 深度学习功能丰富

- **OPTIMIZATION_SUMMARY.md** - 优化总结
  - 优化内容总结
  - 优化统计

- **PROJECT_OPTIMIZATION.md** - 项目优化计划
  - 优化计划
  - 优化目标

- **PROJECT_OPTIMIZATION_COMPLETE.md** - 项目优化完成报告
  - 完成的优化项
  - 优化结果

- **ISSUES_AND_OPTIMIZATION.md** - 问题与优化
  - 发现的问题
  - 优化方案

### 项目报告 (`project/`)

- **PROJECT_ORGANIZATION.md** - 项目组织说明
  - 项目重组记录
  - 目录结构说明

- **PROJECT_OVERVIEW.md** - 项目总览
  - 项目概述
  - 功能说明

- **ARCHITECTURE.md** - 架构文档
  - 系统架构
  - 设计说明

- **REORGANIZATION.md** - 脚本重组说明
  - 脚本重组记录

- **SCRIPT_STRUCTURE.md** - 脚本结构说明
  - 脚本目录结构

### 代码报告 (`code/`)

- **CODE_OPTIMIZATION.md** - 代码优化建议
  - 发现的优化点
  - 优化建议

- **CODE_OPTIMIZATION_SUMMARY.md** - 代码优化总结
  - 已完成的优化
  - 优化统计

### 统计数据 (`statistics/`)

此目录用于存放未来生成的统计数据文件（JSON、CSV等格式）。

## 使用说明

1. **生成新报告时**：请根据报告类型放入对应的子目录
2. **报告命名**：使用清晰的命名，包含日期或版本号
3. **更新索引**：更新 `docs/INDEX.md` 以包含新报告
