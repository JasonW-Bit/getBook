# 改写风格说明

## 所有可用风格

### 基础风格（8种）

#### 1. 现代
- **特点**：现代简洁的语言风格，保持原样或轻微调整
- **适用**：现代都市小说、现实主义作品
- **示例**：`--style=现代`

#### 2. 古典
- **特点**：古典文雅的文言文风格
- **转换**：的→之，了→矣，吗→乎，呢→焉
- **适用**：古代背景小说、历史小说
- **示例**：`--style=古典`

#### 3. 简洁
- **特点**：简洁明了的表达方式，去除冗余
- **转换**：简化长句，去除重复标点
- **适用**：需要精简表达的文本
- **示例**：`--style=简洁`

#### 4. 华丽
- **特点**：华丽优美的辞藻，增加修饰
- **转换**：添加形容词，增加修饰词
- **适用**：需要文采飞扬的作品
- **示例**：`--style=华丽`

#### 5. 悬疑
- **特点**：悬疑紧张的氛围
- **转换**：增加紧张氛围的描写
- **适用**：悬疑小说、推理小说
- **示例**：`--style=悬疑`

#### 6. 浪漫
- **特点**：浪漫温馨的描写
- **转换**：增加浪漫元素的描写
- **适用**：爱情小说、言情小说
- **示例**：`--style=浪漫`

#### 7. 幽默
- **特点**：幽默风趣的表达
- **转换**：增加幽默元素
- **适用**：轻松幽默的小说
- **示例**：`--style=幽默`

#### 8. 严肃
- **特点**：严肃庄重的语调
- **转换**：增加段落间距，正式表达
- **适用**：严肃题材、正剧
- **示例**：`--style=严肃`

### 扩展风格（10种）

#### 9. 科幻
- **特点**：科幻未来感的风格，充满科技元素
- **转换**：增加科技感词汇，未来感表达
- **适用**：科幻小说、未来题材
- **示例**：`--style=科幻`

#### 10. 武侠
- **特点**：武侠小说的风格，充满江湖气息
- **转换**：增加武侠元素，如"轻功"、"抱拳"等
- **适用**：武侠小说、江湖题材
- **示例**：`--style=武侠`

#### 11. 青春
- **特点**：青春活泼的风格，轻松明快
- **转换**：使用青春用语，增加活力
- **适用**：青春小说、校园题材
- **示例**：`--style=青春`

#### 12. 都市
- **特点**：现代都市生活的风格
- **转换**：增加都市生活场景描写
- **适用**：都市小说、现代生活题材
- **示例**：`--style=都市`

#### 13. 古风
- **特点**：古代文雅的风格
- **转换**：使用古风词汇，文雅表达
- **适用**：古代背景、古风小说
- **示例**：`--style=古风`

#### 14. 诗化
- **特点**：诗意化的表达，充满文学美感
- **转换**：增加诗意描写，分行表达
- **适用**：文学性强的作品
- **示例**：`--style=诗化`

#### 15. 口语
- **特点**：口语化的表达，贴近日常对话
- **转换**：使用口语词汇，更贴近生活
- **适用**：需要口语化表达的作品
- **示例**：`--style=口语`

#### 16. 正式
- **特点**：正式书面语的风格
- **转换**：使用正式词汇，书面语表达
- **适用**：正式场合、官方文档风格
- **示例**：`--style=正式`

#### 17. 网络
- **特点**：网络用语风格，轻松活泼
- **转换**：使用网络流行语
- **适用**：网络小说、轻松题材
- **示例**：`--style=网络`

#### 18. 文艺
- **特点**：文艺范的风格，充满文学气息
- **转换**：增加文艺描写，分行表达
- **适用**：文艺小说、文学性作品
- **示例**：`--style=文艺`

## 风格对比示例

### 原文
> 陈旭说："好的，我明白了。"

### 不同风格改写

- **现代**：陈旭说："好的，我明白了。"
- **古典**：陈旭曰："善，吾明之矣。"
- **简洁**：陈旭说："明白了。"
- **华丽**：陈旭如此说道："好的，我完全明白了。"
- **悬疑**：陈旭说："好的，我明白了。"【气氛紧张】
- **浪漫**：陈旭说，眼中闪烁着温柔的光芒："好的，我明白了。"
- **幽默**：陈旭说："好的，我明白了。"【有趣的是】
- **科幻**：陈旭通过通讯器说道："好的，我明白了。"
- **武侠**：陈旭抱拳说道："好的，我明白了。"
- **青春**：陈旭说："好的，我超级明白了。"
- **古风**：陈旭曰："善，吾明之焉。"
- **诗化**：陈旭说："好的，我明白了。"（分行表达）
- **口语**：陈旭说："好的，我挺明白了。"
- **正式**：陈旭表示："好的，我明白了。"
- **网络**：陈旭说："好的，我超明白了。"
- **文艺**：陈旭轻声说道："好的，我明白了。"（分行表达）

## 使用建议

1. **组合使用**：可以结合视角转换和姓名替换
   ```bash
   python3 rewrite_novel.py novel.txt --style=悬疑 --perspective=第三人称 --replace-names
   ```

2. **AI增强**：使用AI可以获得更好的改写效果
   ```bash
   python3 rewrite_novel.py novel.txt --use-ai --style=科幻
   ```

3. **风格选择**：
   - 根据小说类型选择对应风格
   - 悬疑小说 → 悬疑风格
   - 爱情小说 → 浪漫风格
   - 武侠小说 → 武侠风格
   - 科幻小说 → 科幻风格

---

## 注意事项

- 传统方法的风格转换是规则基础的，效果有限
- 使用AI可以获得更自然、更准确的风格转换
- 某些风格（如科幻、武侠）在传统方法下效果较简单
- 建议对重要作品使用AI进行改写
